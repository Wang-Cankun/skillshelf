# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

By participating, you agree to uphold a respectful, harassment-free, and inclusive
environment for everyone, regardless of background or identity.

## Scope

This applies within all project spaces (issues, pull requests, discussions) and in public
spaces when an individual is representing the project.

## Reporting

Report concerns by opening a confidential
[GitHub Security Advisory](https://github.com/Wang-Cankun/skillshelf/security/advisories/new)
or a private issue to the maintainers. All reports are reviewed and handled in good faith.

The full text and enforcement guidelines are available at the link above.
