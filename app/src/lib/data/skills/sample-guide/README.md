# Sample Guide — README

This is a sample skill bundled with SkillShelf to demonstrate how a skill with
multiple files appears in the detail drawer. It performs no real work.

## Files

- `SKILL.md` — the skill definition and workflow.
- `references/api.md` — example of a reference doc with a small API table.
- `references/design-theory.md` — example of a longer-form rationale doc.

## Usage

Invoke with `/sample-guide` or ask the assistant to "write a how-to guide".
